EXTRAVAGANT CREATIVE DIRECTION

Transform the website into an extravagant, flamboyant and highly interactive digital festival experience.

The design should feel like entering a premium Bangla Mela rather than browsing a conventional community website.

It must be:

• Visually spectacular
• Colour-rich
• Luxurious
• Energetic
• Celebratory
• Culturally expressive
• Highly interactive
• Memorable
• Modern
• Sponsor-ready

The website should combine the atmosphere of:

• A major cultural festival
• A luxury event launch
• A theatrical stage production
• A modern interactive digital experience
• Premium Bangladeshi decorative art
• High-end corporate sponsorship presentation

Do not create a plain charity website, ordinary council website, basic community template or generic corporate landing page.

VISUAL PERSONALITY

Use a bold visual identity built around:

• Deep emerald green
• Rich Bangladesh red
• Metallic gold
• Warm ivory
• Jewel tones
• Glowing amber festival lights
• Occasional turquoise, magenta and saffron accents

Use bright colours confidently, but maintain a strong visual hierarchy.

Avoid making every section equally colourful. Alternate between:

• Dramatic dark-green sections
• Bright cream sections
• Cinematic photography
• Rich red sections
• Gold-accented feature areas
• Immersive full-screen sections

Use layered gradients, light textures, blurred colour glows, soft glass effects and subtle metallic details.

The design should feel flamboyant and luxurious without becoming messy, childish or difficult to read.

HERO EXPERIENCE

Create a spectacular full-screen animated hero.

The hero should feature:

• A cinematic Bangla Mela festival photograph or video-style background
• Animated strings of warm festival lights
• Green triangular bunting with red circles moving gently
• Floating gold particles
• Subtle light flares
• Decorative Bengali-inspired patterns
• Layered parallax depth
• Animated headline reveal
• Animated event information
• Prominent interactive call-to-action buttons

The main headline should enter with a bold theatrical animation:

WALSALL’S FIRST EVER
BANGLA MELA 2026

Use a staggered letter or word reveal rather than displaying all text instantly.

Allow “BANGLA MELA 2026” to appear larger, brighter and more dramatic than the supporting text.

Add a subtle animated gold shimmer moving across the main heading.

The official logo should animate gently into place using opacity, scale and upward movement.

Do not rotate, distort, redraw or break apart the official logo.

Add an animated scroll indicator at the bottom of the hero.

OPTIONAL HERO INTRO

When the website first loads, show a short two-to-three-second branded introduction:

1. Deep-green screen
2. Fine gold decorative lines animate outward
3. Official logo fades and scales into view
4. A soft red circular glow appears behind it
5. The phrase “Culture, Community & Togetherness” appears
6. The screen transitions smoothly into the main hero

Include a “Skip Intro” option.

Do not make visitors watch the introduction again on every page.

Use session storage so it appears only once per browsing session.

ANIMATION STYLE

Use elegant, high-quality animations throughout the website.

Animations should feel:

• Smooth
• Cinematic
• Layered
• Purposeful
• Premium
• Responsive
• Festival-inspired

Use:

• Fade-and-rise entrances
• Masked image reveals
• Curved wipe transitions
• Staggered text animation
• Number count-up animation
• Card hover transformations
• Soft parallax movement
• Animated underlines
• Decorative line drawing
• Floating particles
• Moving light reflections
• Smooth accordion transitions
• Animated tab changes
• Interactive package comparisons
• Image depth effects
• Cursor-responsive highlights on desktop

Do not use:

• Constant bouncing
• Aggressive shaking
• Excessive spinning
• Fast flashing
• Random animation
• Animations that block content
• Animations that make the website feel like a game
• Effects that cause motion sickness

All animations must respect the user’s reduced-motion preference.

SCROLL EXPERIENCE

Design the homepage as a continuous interactive journey.

Each major section should reveal itself as the visitor scrolls.

Use smooth transitions between sections such as:

• Curved green-to-cream wipes
• Gold brush-stroke reveals
• Layered fabric-inspired movement
• Decorative pattern transitions
• Cinematic image reveals
• Soft colour-gradient changes

Use sticky sections selectively.

For example:

• Keep the sponsorship heading fixed while package cards move horizontally
• Keep event statistics visible while their descriptions animate in
• Pin the “Before, During and After” timeline while each phase becomes active
• Use a sticky event image while the About content scrolls beside it

Do not use sticky scrolling for every section.

INTERACTIVE EVENT HIGHLIGHTS

Turn the event highlights into large animated feature cards.

Cards should include:

• Full-bleed photography
• Large titles
• Short descriptions
• Animated gold borders
• Gradient overlays
• Hover image movement
• Expandable detail panels
• Touch-friendly mobile interactions

On desktop, hovering over a card should:

• Slightly enlarge the image
• Raise the card
• Reveal a gold light sweep
• Display an arrow or “Explore” label

On mobile, tapping should reveal the same information without relying on hover.

EVENT STATISTICS

Create a dramatic animated statistics section.

Use large oversized numbers such as:

3,000–5,000
25,000+
50+
25+

Animate each number when it enters the viewport.

Place the statistics over a dark-green or cinematic background with:

• Subtle gold particle effects
• Animated lines
• Gentle background glow
• Decorative cultural motifs

Allow each statistic to expand or reveal additional information when selected.

SPONSORSHIP EXPERIENCE

Make the sponsorship section one of the most visually impressive areas of the website.

Create luxurious interactive package cards for:

• Platinum
• Gold
• Silver
• Bronze
• Community Partner

Each package should have a distinct premium visual treatment.

PLATINUM

• Deep green and platinum metallic styling
• Animated edge glow
• Largest card
• “Headline Sponsor” ribbon
• Subtle moving shine
• Strongest call-to-action

GOLD

• Gold foil-style accents
• Warm light reflections
• Elegant raised-card effect

SILVER

• Cool metallic highlights
• Refined glass and silver details

BRONZE

• Rich bronze gradient
• Warm metallic border

COMMUNITY PARTNER

• Friendly red-and-green styling
• Warm, inclusive visual treatment

When a package is selected:

• Expand its benefits smoothly
• Highlight the package
• Dim the remaining packages slightly
• Update the enquiry button automatically
• Pre-fill the relevant sponsorship option in the form

Add a visually exciting package comparison tool.

Allow visitors to switch between:

• Card view
• Comparison view

On desktop, package cards may move through an interactive horizontal carousel.

On mobile, use a smooth swipeable card interface.

Do not hide important package information behind animation.

BEFORE, DURING AND AFTER TIMELINE

Turn this section into an interactive animated story.

Create three large phases:

BEFORE THE EVENT
Building Anticipation

DURING THE EVENT
Maximum Visibility

AFTER THE EVENT
Lasting Impact

As the user scrolls:

• The active phase changes
• A progress line animates
• Supporting images transition
• Relevant benefits appear
• The background colour changes subtly
• The phase title enlarges
• Icons animate into position

Use large theatrical typography and strong visual contrast.

Add the statement:

LOCAL ROOTS
REGIONAL STRENGTH
NATIONAL REACH

Reveal each line separately with bold animated typography.

CULTURAL DECORATIVE ELEMENTS

Use culturally inspired decoration throughout the website, including:

• Bengali-inspired geometric patterns
• Decorative arches
• Paisley-inspired linework
• Floral motifs
• Gold ornamental corners
• Green and red bunting
• Festival lanterns
• Fine filigree patterns
• Flowing fabric-inspired curves
• Hand-painted brush textures

These elements must be original decorative interpretations.

Do not copy sacred, religious or culturally sensitive symbols without context.

Keep decorations behind or around content rather than over important text.

Use animated decorative lines that draw themselves into view.

Use lanterns and festoon lights as subtle moving foreground or background elements in selected sections.

Do not repeat the same decorative element everywhere.

IMAGE PRESENTATION

Do not place every image inside a basic rectangle.

Use varied treatments such as:

• Curved image masks
• Arched frames
• Circular cut-outs
• Layered image collages
• Full-width cinematic banners
• Gold-edged portrait frames
• Floating photo cards
• Diagonal image reveals
• Soft blended image transitions
• Overlapping foreground and background images

Images may move slightly with the cursor or scroll to create depth.

Keep movement subtle enough to preserve image quality and usability.

Use different images throughout the website.

Do not repeatedly reuse the same hero image.

• Masonry layout

• Lightbox viewing
• Keyboard navigation
• Swipe controls on mobile
• Image captions

• Fluid transitions between images

Gallery filters:

• Performances

CTA SECTIONS

Make calls to action visually dramatic.

Use full-width animated sections with:

• Oversized headings
• Moving gradient light
• Decorative gold borders
• Festival imagery
• Animated bunting
• Glowing buttons
• Layered patterns

Buttons should feel premium and responsive.

Button interactions may include:

• Soft glow
• Slight upward movement
• Animated arrow
• Colour transition
• Gold shine passing across the surface

Do not use excessive button animation.

FOOTER EXPERIENCE

Create an extravagant deep-green footer with:

• Large official logo
• Gold decorative pattern
• Animated festoon lights or lantern glow
• Event date and venue
• Sponsor call-to-action
• Contact links
• Social icons
• Newsletter form
• Navigation links

Add a final message:

Together We Can Build Stronger Communities and a Brighter Future

Use a subtle animated gold line beneath it.

MICRO-INTERACTIONS

Include refined micro-interactions throughout:

• Navigation underline movement
• Button arrow movement
• Icon response on hover
• Smooth form-field focus
• Animated checkbox confirmation
• Success-state celebration
• Accordion icon rotation
• Card tilt of no more than a few degrees
• Image zoom of no more than 5%
• Animated menu reveal
• Smooth page transitions

After a successful enquiry submission, show a tasteful celebration:

• Small burst of gold particles
• Confirmation tick
• Thank-you message

Do not use loud confetti across the entire screen.

NAVIGATION

Create a luxurious animated navigation menu.

Desktop:

• Sticky transparent header over the hero
• Header becomes deep green after scrolling
• Gold animated underline beneath active links
• Prominent red or gold sponsor button

Mobile:

• Full-screen deep-green menu
• Large elegant navigation labels
• Gold dividing lines
• Subtle decorative patterns
• Smooth staggered link entrance
• Clearly visible close button
• Sponsor call-to-action at the bottom

PAGE TRANSITIONS

Use smooth branded page transitions.

When moving between pages:

• Fade content out
• Show a short green-and-gold transition
• Fade the next page in
• Preserve fast loading and accessibility

Do not use long loading screens between pages.

CURSOR EFFECTS

On desktop only, optionally add a subtle custom cursor treatment.

Examples:

• Soft gold halo
• Small responsive dot
• Slight enlargement over buttons and links

Do not replace the standard pointer with something difficult to use.

Disable custom cursor effects on touch devices.

SOUND

Do not autoplay music or sound.

An optional clearly labelled sound control may be included for a short ambient festival soundscape, but it must remain off by default.

ACCESSIBILITY AND PERFORMANCE

Despite the extravagant style, maintain:

• WCAG 2.2 AA contrast
• Clear readable typography
• Visible keyboard focus
• Keyboard-operable interactions
• Reduced-motion support
• Semantic page structure
• Descriptive image alternative text
• Large touch targets
• Fast mobile performance
• Optimised animation files
• Lazy-loaded images
• No flashing content
• No horizontal overflow
• No animation that prevents visitors from reading or interacting

Use CSS and lightweight animation libraries efficiently.

Prefer transforms and opacity for animations.

Avoid excessive large video backgrounds or heavy effects that cause poor performance.

The site should still achieve strong Lighthouse scores.

FINAL CREATIVE STANDARD

The final website should look like a bespoke, award-quality cultural event website.

It should immediately communicate:

• This is a major landmark celebration
• The event is professionally organised
• The culture is vibrant and proudly represented
• Sponsors will receive high-quality exposure
• Families will experience an exciting festival
• Shongo Shomithi is a credible and ambitious organisation

The result must be extravagant and flamboyant, but still refined, usable, accessible and professional.

Every page should contain at least one memorable visual moment, but no page should feel overcrowded.

Do not tone the design down into a conventional template.
